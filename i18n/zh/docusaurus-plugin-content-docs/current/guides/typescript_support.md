# TypeScript支持 {#typescript-support}

从v2.0版本起，DHTMLX RichText库提供了使用TypeScript定义的能力。TypeScript的内置支持开箱即用。

:::note[注意]
您可以直接在我们的<a href="https://snippet.dhtmlx.com/ziynafp7?tag=richtext" target="_blank">Snippet Tool</a>中试用该功能。
:::

## 使用TypeScript的优势 {#advantages-of-using-typescript}

为什么需要将DHTMLX RichText与TypeScript结合使用？

TypeScript的主要优势在于它可以显著提高开发效率。

在构建应用程序时，类型检查与自动补全相结合，可以帮助您避免潜在错误，使开发过程更加稳健。此外，TypeScript还能为您提供在使用DHTMLX RichText库API时应使用的数据类型信息。
