# API 概览

## RichText 构造函数

~~~js
new richtext.RichText("#root", {
    // 配置参数
});
~~~

**参数**:

- 一个 HTML 容器（例如 CSS 选择器或 DOM 元素）
- 一个配置对象（[参见属性](api/overview/properties_overview.md)）
