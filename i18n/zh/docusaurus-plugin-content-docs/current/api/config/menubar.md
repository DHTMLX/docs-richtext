# menubar

### 描述 {#description}

@short: 可选。启用 RichText 顶部菜单栏

### 用法 {#usage}

~~~jsx {}
menubar?: boolean; 
~~~

### 示例 {#example}

~~~jsx {3}
// 初始化 RichText
new richtext.Richtext("#root", {
    menubar: true
    // 其他配置属性
});
~~~
