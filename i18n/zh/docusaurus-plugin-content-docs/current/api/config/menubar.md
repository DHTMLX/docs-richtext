# menubar

### Description

@short: 可选。启用 RichText 顶部菜单栏

### Usage

~~~jsx {}
menubar?: boolean; 
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    menubar: true
    // other configuration properties
});
~~~
