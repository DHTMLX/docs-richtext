# menubar

### Description

@short: 선택 사항. RichText 상단 메뉴바를 활성화합니다

### Usage

~~~jsx {}
menubar?: boolean; 
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    menubar: true
    // other configuration properties
});
~~~
