# menubar

### 설명 {#description}

@short: 선택 사항. RichText 상단 메뉴바를 활성화합니다

### 사용법 {#usage}

~~~jsx {}
menubar?: boolean;
~~~

### 예제 {#example}

~~~jsx {3}
// RichText 초기화
new richtext.Richtext("#root", {
    menubar: true
    // 기타 설정 속성
});
~~~

**변경 이력:** 이 속성은 v2.0에서 추가되었습니다
