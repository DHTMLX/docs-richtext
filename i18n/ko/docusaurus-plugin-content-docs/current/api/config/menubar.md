# menubar

### 설명

@short: 선택 사항입니다. RichText 상단에 menubar를 활성화합니다.

### 사용법

~~~jsx {}
menubar?: boolean; 
~~~

### 예제

~~~jsx {3}
// RichText 초기화
new richtext.Richtext("#root", {
    menubar: true
    // 기타 구성 속성
});
~~~

**변경 로그:** 이 속성은 v2.0에서 도입되었습니다.
