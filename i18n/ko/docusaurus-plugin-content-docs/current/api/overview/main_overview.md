# API 개요

## RichText 생성자

~~~js
new richtext.RichText("#root", {
    // 구성 매개변수
});
~~~

**매개변수**:

- HTML 컨테이너(예: CSS 선택자 또는 DOM 요소)
- 구성 객체 ([속성 참조](api/overview/properties_overview.md))
