# undo

### Описание

@short: Срабатывает при нажатии кнопки "Отменить" в меню/панели инструментов или при активации через методы Event Bus.

### Использование

~~~jsx {}
"undo": () => boolean | void;
~~~

:::info
Внутренними событиями можно управлять с помощью [**методов Event Bus**](api/overview/event_bus_methods_overview.md)
:::

### Пример

~~~jsx {5-8}
// инициализация RichText
const editor = new richtext.Richtext("#root", {
    // свойства конфигурации
});
// прослушивание события "undo"
editor.api.on("undo", () => {
    console.log("Операция отмены выполнена");
});
~~~

**Журнал изменений:** Это событие было добавлено в версии v2.0
