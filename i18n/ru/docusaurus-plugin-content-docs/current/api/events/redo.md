# redo

### Описание

@short: Срабатывает при нажатии кнопки "Redo" в менюбара/тулбаре или при активации через методы Event Bus.

### Использование

~~~jsx {}
"redo": () => boolean | void;
~~~

:::info
Для управления внутренними событиями можно использовать [**методы Event Bus**](api/overview/event_bus_methods_overview.md)
:::

### Пример

~~~jsx {5-8}
// инициализация RichText
const editor = new richtext.Richtext("#root", {
    // свойства конфигурации
});
// подписка на событие "redo"
editor.api.on("redo", () => {
    console.log("Операция повторения была выполнена");
});
~~~

**Журнал изменений:** Это событие было добавлено в версии 2.0
