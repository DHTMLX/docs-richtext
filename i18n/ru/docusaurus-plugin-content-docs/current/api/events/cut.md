# cut

### Описание

@short: Срабатывает при вырезании выделенного текста

### Использование

~~~jsx {}
"cut": () => boolean | void;
~~~

:::info
Для управления внутренними событиями можно использовать [**методы Event Bus**](api/overview/event_bus_methods_overview.md).
:::

### Пример

~~~jsx {5-8}
// инициализация RichText
const editor = new richtext.Richtext("#root", {
    // свойства конфигурации
});
// подписка на событие "cut"
editor.api.on("cut", () => {
    console.log("Выделенный текст был вырезан");
});
~~~

**Журнал изменений:** Событие было введено в версии v2.0
