# value

### Описание

@short: Необязательно. Определяет начальное содержимое, отображаемое в области редактора RichText.

:::tip
Чтобы задать содержимое в пользовательском формате, используйте встроенный метод [`setValue()`](api/methods/set-value.md).
:::

### Использование

~~~jsx {}
value?: string; 
~~~

### Пример

~~~jsx {2}
new richtext.Richtext("#root", {
    value: "<h1>some value</h1>" // задаёт значение по умолчанию (формат HTML)
    // другие свойства конфигурации
});
~~~

**История изменений:** Это свойство было добавлено в версии 2.0
