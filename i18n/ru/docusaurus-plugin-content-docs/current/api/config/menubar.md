# menubar

### Описание {#description}

@short: Необязательный. Включает верхний menubar RichText

### Использование {#usage}

~~~jsx {}
menubar?: boolean;
~~~

### Пример {#example}

~~~jsx {3}
// инициализация RichText
new richtext.Richtext("#root", {
    menubar: true
    // другие свойства конфигурации
});
~~~

**История изменений:** Свойство добавлено в v2.0
