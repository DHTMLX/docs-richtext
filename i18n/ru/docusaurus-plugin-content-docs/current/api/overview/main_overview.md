# Обзор API

## Конструктор RichText

~~~js
new richtext.RichText("#root", {
    // параметры конфигурации
});
~~~

**Параметры**:

- HTML контейнер (например, CSS селектор или DOM элемент)
- объект конфигурации ([см. свойства](api/overview/properties_overview.md))
