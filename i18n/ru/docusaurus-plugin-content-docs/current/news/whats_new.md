## Версия 2.0.5 

Выпущена 6 марта 2026 года

### Исправления

- Неправильная обработка ввода через IME
- Неправильная обработка ввода на основе композиции (например, китайский, японский, корейский на рабочем столе)
- Свойства `margin-left` и `line-height` игнорируются при разборе HTML
- Строчные элементы наследуют свойства от блочных родителей
- Невозможно скопировать текст между двумя экземплярами редактора
- Некорректная обработка выделения при наличии двух (или более) активных экземпляров редактора
- Курсор неправильно позиционируется, если единственным содержимым абзаца является изображение
- Бесконечное оборачивание элементов подстрочного и надстрочного индекса
- При вставке изображения оно встраивается как строчный элемент вместо предварительной загрузки
- Пустые строчные блоки нельзя удалить в определённых позициях (первая или последняя строка)
- Редактор не прокручивается к курсору при вводе
- Неправильная обработка ввода на Android
- Некорректная обработка выделения для элементов HR
- Некорректное отображение при пустых начальных значениях
- Копирование контента может завершаться ошибкой, если он содержит самозакрывающиеся блоки
- Chrome: невозможно вставить эмодзи из контекстного меню
- Firefox: выделение всего текста (CTRL+A) обрабатывается некорректно

## Версия 2.0.4

Выпущена 15 октября 2025 года

### Исправления

- Некорректное выделение диапазона для абзацев
- Всплывающие окна ссылок не отображаются рядом с изображениями-ссылками
- Начальные значения панели инструментов не соответствуют фактическому содержимому

## Версия 2.0.3

Выпущена 27 августа 2025 года

### Исправления

- Клик по горизонтальной линии вызывает ошибку скрипта
- Некорректная высота строки по умолчанию
- Стили контента содержат дублирующиеся селекторы
- Блоки простого текста игнорируют изменения стиля при разборе HTML
- Парсер игнорирует экранированные названия шрифтов в HTML
- Парсер игнорирует свойства `margin-left` и `line-height` в HTML

### Новые возможности

- **Новое поколение рендеринга текста**  
  Более плавный, быстрый и точный рендеринг текста на базе нового движка

- **Гибкая настройка панели инструментов**  
  Полный контроль над панелью инструментов:  
  - Задавайте [отдельные элементы управления](guides/configuration.md/#default-toolbar-controls) и их порядок
  - Добавляйте [пользовательские элементы управления](guides/configuration.md/#custom-toolbar-controls)

- **Опциональная [строка меню](api/config/menubar.md)**  
  Включите классический интерфейс с меню в верхней части редактора

- **Расширенный [режим документа](guides/configuration.md/#layout-modes)**  
  Добавлена поддержка различных размеров документа

- **Поддержка изображений**  
  - Вставляйте изображения с [возможностью загрузки](api/config/image-upload-url.md)  
  - [Изменяйте размер изображений](api/events/resize-image.md) в интерактивном режиме  

- **Улучшенная работа со ссылками**  
  Переработанные [всплывающие взаимодействия](api/events/show-popup.md) для удобства использования

- **Новые инструменты форматирования**  
  - Выравнивание текста: **по ширине**  
  - Вставка [горизонтальных линий](api/events/insert-line.md)  
  - [Увеличение отступа](api/events/indent.md) / [уменьшение отступа](api/events/outdent.md)  
  - Установка [высоты строки](api/events/set-line-height.md)  
  - Применение [подстрочного](api/events/subscript.md) / [надстрочного](api/events/superscript.md) индекса

- **Управление списками**  
  Удобная [вставка и управление списками](api/events/insert-list.md) в контенте

- **Импорт/Экспорт и Печать**  
  - [Импорт](api/events/import.md) файлов DOCX
  - [Экспорт](api/events/export.md) контента в DOCX или PDF
  - [Печать](api/events/print.md) документов прямо из редактора

- **Горячие клавиши**  
  Расширенная поддержка стандартных сочетаний клавиш для форматирования и редактирования
