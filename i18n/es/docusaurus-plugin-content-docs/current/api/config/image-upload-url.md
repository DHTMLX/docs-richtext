# imageUploadUrl

### Descripción {#description}

@short: Opcional. Especifica la URL que se utilizará para la carga de imágenes

### Uso {#usage}

~~~jsx {}
imageUploadUrl?: string;
~~~

### Ejemplo {#example}

~~~jsx {3}
// inicializar RichText
new richtext.Richtext("#root", {
    imageUploadUrl: "some URL"
    // otras propiedades de configuración
});
~~~
