# imageUploadUrl

### Description

@short: Optional. Specifies the URL which will be used for image upload

### Usage

~~~jsx {}
imageUploadUrl?: string;
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    imageUploadUrl: "some URL"
    // other parameters
});
~~~

**Change log:** The property was added in v2.0
