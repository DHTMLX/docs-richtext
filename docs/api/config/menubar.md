# menubar

### Description {#description}

@short: Optional. Enables the top menubar of RichText

### Usage {#usage}

~~~jsx {}
menubar?: boolean; 
~~~

### Example {#example}

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    menubar: true
    // other configuration properties
});
~~~

**Change log:** The property was added in v2.0
