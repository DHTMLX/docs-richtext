# menubar

### Description

@short: Optional. Turns on the top menubar in RichText.

### Usage

~~~jsx {}
menubar?: boolean; 
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    menubar: true
    // other configuration properties
});
~~~

**Change log:** This property was introduced in v2.0
