# fullscreenMode

### Description

@short: Optional. Enables the RichText fullscreen mode

### Usage

~~~jsx {}
fullscreenMode?: boolean; 
~~~

### Default config

~~~jsx
fullscreenMode: false;
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    fullscreenMode: true
    // other parameters
});
~~~

**Change log:** The property was added in v2.0
