# fullscreenMode

### Description

@short: Optional. Turns on the RichText fullscreen mode

### Usage

~~~jsx {}
fullscreenMode?: boolean; 
~~~

### Default config

~~~jsx
fullscreenMode: false;
~~~

### Example

~~~jsx {3}
// initialize RichText
new richtext.Richtext("#root", {
    fullscreenMode: true
    // other configuration properties
});
~~~

**Change log:** This property was introduced in v2.0
