# fullscreen

### Description

@short: Optional. Enables the editor fullscreen mode

### Usage

~~~jsx {}
fullscreen?: boolean; 
~~~

### Default config

~~~jsx
fullscreen: false
~~~

### Example

~~~jsx {2}
new richtext.Richtext("#root", {
    fullscreen: true
    // other parameters
});
~~~

**Change log:** The property was added in v2.0
