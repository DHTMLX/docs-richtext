# API overview

## RichText constructor

~~~js
new richtext.RichText("#root", {
    // configuration parameters
});
~~~

**Parameters**:

- an HTML container (e.g., CSS selector or DOM element)
- a configuration object ([see properties](api/overview/properties_overview.md))
