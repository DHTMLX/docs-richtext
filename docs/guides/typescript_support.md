# TypeScript support

DHTMLX RichText v2.0 and later includes built-in TypeScript definitions. TypeScript support works out of the box.

:::note
You can try out the functionality right in our <a href="https://snippet.dhtmlx.com/ziynafp7?tag=richtext"  target="_blank">Snippet Tool</a>.
:::

## Advantages of TypeScript

TypeScript increases development efficiency by catching type errors early and providing autocompletion. Type information is especially useful when working with the DHTMLX RichText API.
